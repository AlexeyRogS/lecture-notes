2.2 Gaussian Wave Packets

We will now look at a concrete example of a valid state for a free particle constructed as a wave packet and study its properties to get a general idea of how wavefunctions behave in quantum mechanics. As discussed in section 2.1.2 above, this begins with choosing a set of wavenumbers and their amplitudes A⁢(k). We will start with constructing the wave packet at t=0, then we will discuss how it evolves with time.

2.2.1 Choosing an Amplitude A⁢(k)

It would be nice to choose A⁢(k) to be a Gaussian shape, because then the Fourier transform ψ⁢(x,0) will also be a Gaussian, and Gaussians are nice to work with.

Recall that a Gaussian function has the general form

aπ⁢e−(x−c)2b2, (2.19)

which is a bell curve centered at c, with height aπ, width b, and area a⁢b. This means if we want our wavenumber probability density to be a Gaussian centered at some wavenumber k0 with width Δ⁢k, we should choose a=1Δ⁢k so that it will be normalised.

A⁢(k)2=1Δ⁢k⁢π⁢e−(k−k0)2Δ⁢k2. (2.20)

The width Δ⁢k is the distance from the centre at e−1 of the maximum height. It is a rough measure of the range of wavenumbers of the plane waves that make up the wave packet. Note that Δ⁢p=ℏ⁢Δ⁢k, so the width Δ⁢k is directly related to the uncertainty in momentum.

2.2.2 Calculating the Wavefunction ψ⁢(x,0)

Using the Fourier transform given by equation 2.14, the wavefunction at t=0 is given by

ψ⁢(x,0) =12⁢π⁢∫−∞∞A⁢(k)⁢ei⁢k⁢x⁢d⁡x (2.21)
=1Δ⁢k⁢π⁢2⁢π⁢∫−∞∞e−(k−k0)22⁢Δ⁢k2⁢ei⁢k⁢x⁢d⁡k, (2.22)

where in the second line we inserted the square root of equation 2.20 for A⁢(k). To integrate this we use the standard result for Gaussian integrals:

∫−∞∞e−α⁢q2⁢e−β⁢q⁢d⁡q=πα⁢eβ24⁢α. (2.23)

With q=k−k0 (so d⁡q=d⁡k), we get

ψ⁢(x,0) =1Δ⁢k⁢π⁢2⁢π⁢∫−∞∞e−q22⁢Δ⁢k2⁢ei⁢(q+k0)⁢x⁢d⁡q (2.24)
=1Δ⁢k⁢π⁢2⁢π⁢ei⁢k0⁢x⁢∫−∞∞e−q22⁢Δ⁢k2⁢ei⁢q⁢x⁢d⁡q. (2.25)

We can now use the standard integral with α=12⁢Δ⁢k2 and β=−i⁢x to get

ψ⁢(x,0) =2⁢π⁢Δ⁢k2Δ⁢k⁢π⁢2⁢π⁢ei⁢k0⁢x⁢e−Δ⁢k2⁢x22 (2.26)
=Δ⁢kπ⁢ei⁢k0⁢x⁢e−x2⁢Δ⁢k22. (2.27)

What is the probability density at t=0? We can calculate this directly by taking the magnitude square:

|ψ⁢(x,0)|2=Δ⁢kπ⁢e−x2⁢Δ⁢k2=Δ⁢kπ⁢e−x2Δ⁢x02, (2.28)

where we have defined Δ⁢x0=1Δ⁢k as the width at t=0. So the probability density is also a Gaussian, this time centered at x=0, with width Δ⁢x0=1Δ⁢k.

Notice that Δ⁢x0⁢Δ⁢k=1, which can be expressed in term of momentum via the de Broglie relation:

Δ⁢x0⁢Δ⁢p=ℏ. (2.29)

This is a manifestation of the uncertainty principle. The factor of 1 arises from our definition of width of a Gaussian. Notice that the width of the position probability density is inversely proportional to the width of the wavenumber probability density. This reflects the fact that due to the uncertainty principle, if the uncertainty in momentum Δ⁢k is small then the uncertainty in position Δ⁢x0 must be wide and vice versa.

2.2.3 Time-evolution of the Gaussian Wave Packet

As we found in section 2.1.2, the time evolution of our wave packet is given by

ψ⁢(x,t)=12⁢π⁢∫−∞∞A⁢(k)⁢ei⁢(k⁢x−ω⁢t)⁢d⁡k. (2.30)

To evaluate this, we need to keep in mind that ω is a function of k through the Einstein and de Broglie relations. Specifically, for a free particle only we have E=p22⁢m, which gives

E =p22⁢m (2.31)
ℏ⁢ω =ℏ2⁢k22⁢m (2.32)
⟹ω⁢(k) =ℏ⁢k22⁢m. (2.33)

This is our dispersion relation for matter waves. Note that the ω is not proportional to k, so matter waves are dispersive. This means that different wavelength will travel at different speeds. The phase velocity is given by

vph⁢(k)=ωk=ℏ⁢k2⁢m, (2.34)

so shorter wavelengths (larger k) travel faster. This makes physical sense as large k corresponds to large momentum. Meanwhile the group velocity for a wave packet centered at k0 is

vgr⁢(k)=d⁡ωd⁡k|k0=ℏ⁢k02⁢m=2⁢vph⁢(k0). (2.35)

This is the speed that the envelope of the wave packet will travel at. Combining these ideas together, we expect that the wave packet will be smudged out over time, as some wavelengths in it travel faster and some slower.

Let’s see if we can show this explicitly. Substituting the square root of equation 2.20 for A⁢(k) and equation 2.33 for ω⁢(k) into the integral, we get

ψ⁢(x,t)=1Δ⁢k⁢π⁢2⁢π⁢∫−∞∞e−(k−k0)22⁢Δ⁢k2⁢ei⁢(k⁢x−ℏ⁢k22⁢m⁢t)⁢d⁡k. (2.36)

We would like to use the standard integral 2.23 for Gaussians again. Making the same substitution as in section 2.2.2 above (q=k−k0) gives

ψ⁢(x,t) =1Δ⁢k⁢π⁢2⁢π⁢∫−∞∞e−q22⁢Δ⁢k2⁢ei⁢(q+k0)⁢x⁢e−i⁢ℏ⁢(q+k0)22⁢m⁢t⁢d⁡q (2.37)
=1Δ⁢k⁢π⁢2⁢π⁢∫−∞∞e−q22⁢Δ⁢k2⁢ei⁢q⁢x⁢ei⁢k0⁢x⁢e−i⁢ℏ⁢q22⁢m⁢t⁢e−i⁢ℏ⁢q⁢k0m⁢t⁢e−i⁢ℏ⁢k022⁢m⁢t⁢d⁡q. (2.38)

We can simplify the notation a bit by defining ω0=ℏ⁢k022⁢m, the frequency of the state with wavenumber k0, and using the group velocity vgr⁢(k0)=ℏ⁢k0m. The fifth and sixth exponentials can be rewritten in terms of these quantities, and then the third and fifth exponentials can be taken out of the integral to get

ψ⁢(x,t)=1Δ⁢k⁢π⁢2⁢π⁢ei⁢(k0⁢x−ω0⁢t)⁢∫−∞∞e−(12⁢Δ⁢k2+i⁢ℏ2⁢m⁢t)⁢q2⁢e−i⁢(vgr⁢(k0)⁢t−x)⁢q⁢d⁡q. (2.39)

Now we can use the standard integral with

α =12⁢Δ⁢k2+i⁢ℏ2⁢m⁢t (2.40)
β =i⁢(vgr⁢(k0)⁢t−x), (2.41)

to get

ψ⁢(x,t) =1Δ⁢k⁢π⁢2⁢π⁢π12⁢Δ⁢k2+i⁢ℏ2⁢m⁢t⁢ei⁢(k0⁢x−ω0⁢t)⁢e−(x−vgr⁢(k0)⁢t)24⁢(12⁢Δ⁢k2+i⁢ℏ2⁢m⁢t) (2.42)
=Δ⁢kπ⁢(1+i⁢ℏ⁢Δ⁢k2⁢tm)⁢ei⁢(k0⁢x−ω0⁢t)⁢e−Δ⁢k2⁢(x−vgr⁢(k0)⁢t)22⁢(1+i⁢ℏ⁢Δ⁢k2⁢tm). (2.43)

Calculating the probability density, we get

|ψ⁢(x,t)|2=Δ⁢kπ⁢1+ℏ2⁢Δ⁢k4⁢t2m2⁢e−Δ⁢k2⁢(x−vgr⁢(k0)⁢t)21+ℏ2⁢Δ⁢k4⁢t2m2. (2.44)

This is quite a complicated expression, but there are a few key things we can pick out that tell us in simple terms how the wave packet behaves. The structure of the probability density is basically

|ψ⁢(x,t)|2=C⁢(t)⁢e−(x−x0⁢(t))2Δ⁢x⁢(t)2, (2.45)

so we see that the height, width, and centre of the wave packet all change with time while retaining a Gaussian form.

Comparing the two expressions, the centre of the wave packet x0 evolves as

x0⁢(t)=vgr⁢(k0)⁢t, (2.46)

so the position of the peak of the wavepacket moves to the right with a constant speed, the speed of the central wavenumber (or equivalently, the speed of the envelope). This makes sense, there are no forces acting on the particle so its speed stays constant with time.

The width of the wave packet is

Δ⁢x⁢(t)=1Δ⁢k⁢1+ℏ2⁢Δ⁢k4⁢t2m2=Δ⁢x0⁢1+ℏ2⁢Δ⁢k4⁢t2m2, (2.47)

where we have reintroduced Δ⁢x0=1Δ⁢k, the width of the wave packet at t=0 from before. Thus the width Δ⁢x increases over time just like we predicted.

The height has the form

C⁢(t)=1π⁢Δ⁢x⁢(t), (2.48)

so since the width increases with time, the height decreases. This makes sense, as to preserve the normalisation of the wave packet the amplitude must decrease as it spreads out.

Notice that in the limit t→0, we get back our previous results for the initial wave packet ψ⁢(x,0). If Δ⁢k is larger at t=0, i.e. if the uncertainty in momentum Δ⁢p is larger, the the wavepacket spreads out more slowly.

Does the uncertainty in momentum Δ⁢p increase over just like Δ⁢x? From equation 2.17, we have

ψ⁢(x,t) =12⁢π⁢ℏ⁢∫−∞∞ϕ⁢(p,t)⁢eiℏ⁢p⁢x⁢d⁡p (2.49)
=12⁢π⁢ℏ⁢∫−∞∞ϕ⁢(p,0)⁢eiℏ⁢(p⁢x−E⁢t)⁢d⁡p. (2.50)

For the free particle, we have E=p22⁢m and so

ψ⁢(x,t)=12⁢π⁢ℏ⁢∫−∞∞ϕ⁢(p,0)⁢e−iℏ⁢p22⁢m⁢t⁢eiℏ⁢p⁢x⁢d⁡p. (2.51)

Hence we have that

|ϕ⁢(p,t)|2=|ϕ⁢(p,0)⁢e−iℏ⁢p22⁢m⁢t|2=|ϕ⁢(p,0)|2, (2.52)

so the momentum probability density remains the same over time and Δ⁢p does not change. This is not a general fact, it only holds for the free particle. We will find that when there are forces present, Δ⁢p does indeed increase over time. In that sense, this is sort of a quantum analogue of Newton’s first law.
